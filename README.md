Symbol Theme
===========
[![image](https://i.cloudup.com/owMC2owNcs.png)](https://cloudup.com/cfgcHouElDv)

Installation
------------
1. Move the folder wp-content/themes/ or upload from your admin area (Plugins > Upload)
2. Enable the theme from the admin panel
